# HomeLens HomeKit Bridge Helper

This helper is intentionally tiny. It publishes one HomeKit camera accessory and one motion sensor using HAP-NodeJS.

The Swift CLI/app remains the supervisor and reliability boundary:

1. Swift owns camera config, Keychain/env secrets, RTSP/ONVIF health checks, and ONVIF event monitoring.
2. This helper owns HomeKit pairing, camera stream sessions, HomeKit characteristics, and the HSV fMP4 prebuffer.
3. Swift sends JSON lines to stdin:

```json
{"type":"motion","active":true}
{"type":"person","active":true}
{"type":"status"}
```

Run directly:

```bash
npm install
HOMELENS_BRIDGE_CONFIG=/path/to/homekit-bridge.json npm start
```

The config file is generated by `homelensctl homekit-config`.

Prebuffer self-test:

```bash
HOMELENS_PASSWORD='camera-password' .build/debug/homelensctl test hsv-prebuffer
```
